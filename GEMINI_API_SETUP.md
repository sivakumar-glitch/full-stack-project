# Get Your FREE Google Gemini API Key (100% FREE - No Payment Required)

## Step 1: Go to Google AI Studio
Visit: https://aistudio.google.com

## Step 2: Create Your API Key
1. Click on **"Get API key"** button
2. Click **"Create API key in new project"**
3. Copy your API key (starts with `sk-...` or similar)

## Step 3: Add to .env File
Open `.env` in the project root and add:
```
LLM_API_KEY=YOUR_COPIED_API_KEY_HERE
```
